This is a dummy project created to learn about react native, this file contains notes about the project.

- Project Structure
    -.expo-shared -> Some internal used information - no clear idea so far
    - assets -> assets being used in the project
    - node_module -> Holds all the third party packages being used under the hood.
    - package.json -> All the dependancies info of our project
    - package-lock.json -> Project and dependancies configurations 
    - babel-config.js -> configurations related to “Transpiler”.
      A tool that transforms code with newer syntax into older code equivalents and this process is called “Transpiling”.
    - app.json -> configurations related to settings and behaviours of the app, build info etc.
    - App.js -> main app coding file
